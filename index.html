hbz reveal.js presentations

We keep hbz presentations based on the reveal.js HTML presentation framework in this repo:

https://github.com/hbz/reveal.js

It is actually a clone of the open source reveal.js repo on GitHub:

https://github.com/hakimel/reveal.js

Presentations are worked on in individual branches, e.g. swib-15:

https://github.com/hbz/reveal.js/tree/swib-15

Presentations are published by copying to a folder in the gh-pages branch:

https://github.com/hbz/reveal.js/tree/gh-pages

They can be browsed at the corresponding GitHub page:

https://github.com/hbz/reveal.js/tree/swib-15
